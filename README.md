## 📌 Project Overview
Capstone assignment extracting and visualizing stock data for **Tesla and GameStop** using two approaches — `yfinance` API for historical stock prices and `BeautifulSoup` web scraping for revenue data. Results are displayed as interactive Plotly dashboards with dual-axis subplots.

**Domain:** Financial Data Analysis  
**Libraries:** yfinance · BeautifulSoup · Plotly · Pandas  
**Platform:** IBM Watson Studio  

## 🚀 Key Features
- `yfinance` API for Tesla & GameStop historical stock prices
- BeautifulSoup web scraping for quarterly revenue data
- Interactive Plotly subplots — stock price + revenue side-by-side
- `make_graph()` function for reusable dashboard creation

## 🎓 Skills Demonstrated
yfinance API · Web scraping with BeautifulSoup · Plotly interactive charts · Pandas DataFrames · Financial data visualization
